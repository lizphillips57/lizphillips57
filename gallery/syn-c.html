Syn-C

July 2023

A music visualizer using p5.js and Gibber, created for a Media Arts course. Drums, chords, and melodies are represented with different shapes, colors, and movements.

There are four different shape classes to utilize: circles, rectangles, waves, and spectra. Each of these elements are completely customizable to react to sound in different ways, including: changing position, size, color, opacity, and/or velocity.

Every instrument can also be modifed using the Gibber library, which can generate unique musical compositions in real time.
